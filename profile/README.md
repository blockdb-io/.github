# BlockDB

Lineage-verified DeFi datasets and real-time on-chain data APIs for quantitative funds, AI teams, and Web3 developers.

BlockDB provides normalized, schema-stable Ethereum and EVM datasets with reorg awareness and cryptographic lineage references. Designed for machine learning pipelines, trading systems, research environments, and production-grade data infrastructure.

---
